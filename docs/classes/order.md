# Order

Von einem Objekt der Klasse Order lassen sich die folgenden Attribute abfragen:

* `typ` -> Typ der Order (`str`)
* `limit` -> Limit der Order (`float`)
* `stueckzahl` -> Aktienanzahl der Order (`int`)
* `orderregel` -> Ob die Order durch eine Orderregel erstellt wurde (`bool`)
* `systembank` -> Ob die Order durch die Systembank erstellt wurde (`bool`)
* `datum` -> Wann die Order beauftragt wurde (`datetime`)
